# DiscreteNaturalNeighbors

Documentation for [DiscreteNaturalNeighbors](https://github.com/benhertzsch/DiscreteNaturalNeighbors.jl). This package implements a discrete natural neighbour interpolation routine for $n$-dimensional data.

## Installation
The DiscreteNaturalNeighbors package can be installed with the Julia package manager.
From the Julia REPL, type `]` to enter the Pkg REPL mode and run:

```julia
pkg> add DiscreteNaturalNeighbors
```

Or, equivalently, via the `Pkg` API:

```julia
julia> import Pkg; Pkg.add("DiscreteNaturalNeighbors")
```
