This package implements the discrete natural neighbor interpolation routine for $n$-dimensional data.

## Installation
The *DiscreteNaturalNeighbors.jl* package can be installed with the Julia package manager.
From the Julia REPL, type `]` to enter the Pkg REPL mode and run:

```julia
pkg> add DiscreteNaturalNeighbors
```

Or, equivalently, via the `Pkg` API:

```julia
julia> import Pkg; Pkg.add("DiscreteNaturalNeighbors")
```
